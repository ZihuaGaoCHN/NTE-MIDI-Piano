一个在异环中使用 MIDI 键盘弹奏钢琴的工具。\
A tool that allows you to use a MIDI keyboard to play the piano in Neverness To Everness.

## 如何使用 | How to use
使用 USB 连接一个 MIDI 设备，如 MIDI 键盘、电钢琴或者合成器，在设置中选择 MIDI 设备，然后在主页启动。\
Connect a MIDI device, such as a MIDI keyboard, digital piano or synthesizer via USB, then select a MIDI device in Settings, then turn on the switch in Home.
